# Subscriber Management Client built with Vite + Vue 3 + TailwindCSS 4

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Resource pages
* `/subscribers`
* `/fields`
